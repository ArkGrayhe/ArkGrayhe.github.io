专业学习|环境配置日记

Posted on 2022-11-20 In Tech , 学习

算是逐渐了解电脑君的身体和心灵的过程。

11-20：

  • 因为conda目录下如同屎山，清理了电脑内存，该卸载的都卸载了。重新装了conda，从零开始。体会是，永远不要直接搬来别人的方法，否则连conda是什么都摸不清。参考Anaconda Documentation — Anaconda documentation 与:: Anaconda.org。还好有hexo经验，其实上手算快。
  • 由于advanced options没有全选，故而修改了环境变量。参考文章Anaconda 介绍、安装与配置 - 哔哩哔哩 (bilibili.com) 学习了修改系统环境变量的方法，参考文章(24条消息) Windows10下Anaconda环境的安装与配置_practical_sharp的博客-CSDN博客 解决了cmd中无法测试出conda版本的问题。
  • 然而cmd中依然无法找到python路径。参考(24条消息) 安装Anaconda后，cmd输入python没有结果_tprtprtpr的博客-CSDN博客 找到大概的问题。由于删除不掉原有路径，把anaconda的路径放置最前。但不知道这里是否会有隐患。 成功找到python。
  • Jupyter Project Documentation — Jupyter Documentation 4.1.1 alpha documentation 笔记本学习文档。
  • 在conda中配置opencv。参考(24条消息) windows环境下的Anaconda安装与OpenCV机器视觉环境搭建_iracer的博客-CSDN博客_anaconda opencv安装 lesson learnt:随便在浏览器上搜索问题是很蠢的，谁能保证推荐机制符合你的诉求呢？尤其是搜到一堆老文章，根本不适用于我们这个领域。请直接在csdn或StackOverflow上搜索。
  • 顺便，发现复制google链接无法自动生成相应格式。之后去搞一下。
  • 今天的配置到此为止。。
